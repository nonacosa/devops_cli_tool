## 英商 devops cli 工具     


 
## 如何安装？

```shell
npm install zing-git -g
```

或者 clone 代码，本地安装

```shell
npm install . -g
```

使用：

```
zgit --help # zgit -h
```    

定期更新，通知大家 update 即可。
